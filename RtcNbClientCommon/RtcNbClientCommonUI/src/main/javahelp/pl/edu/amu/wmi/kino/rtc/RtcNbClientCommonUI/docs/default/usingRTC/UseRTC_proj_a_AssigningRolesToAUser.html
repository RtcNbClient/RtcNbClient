Assigning roles to a user

Important: Modifying project areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
You can assign one or more roles to each user, based on roles that are defined in the process configuration.
Before you begin
You must belong to the JazzAdmins repository group to perform this task.
About this task
To assign roles to a user in a project area or a team area:
  1. In the Admin Web interface, click the Project Area Management tab.
  2. Select a project area from the list of active project areas.
  3. In the Members section, click the Process Roles icon in the entry for a user.
  4. In the Available Roles list, select the role or roles to assign to the user. Click Add.
  5. Click Finish.
  6. Click Save on the project area page to save the changes to the project area.
