Finding work items

A work item is a way of keeping track of the tasks and issues that your team needs to address during the development cycle.

The primary method for finding work items is to run queries against the repository. Queries retrieve work items whose attributes contain values that match those specified in the query. For example, you might run a query that retrieves all work items whose state attribute is Open.

About this task

Rational Team Concert™ provides a set of predefined queries, and you can create additional queries. You can retrieve a specific work item by entering its ID, and you can search for an item by entering a text string. You can also find work items from the plan.

Using a predefined query

Jazz™ provides a set of queries that you can use to retrieve work items that meet a common set of conditions.

Creating a query

Use a query to retrieve a set of work items that meet specific conditions.
