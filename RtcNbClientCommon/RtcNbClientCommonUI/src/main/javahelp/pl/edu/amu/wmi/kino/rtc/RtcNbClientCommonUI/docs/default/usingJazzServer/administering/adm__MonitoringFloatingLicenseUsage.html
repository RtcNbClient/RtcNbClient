Monitoring floating license usage

You can monitor floating license usage through the administrative Web interface.
Before you begin
You must be logged in to the Admin Web UI and be a member of the JazzAdmins group.
About this task
To monitor floating license usage, complete any of the following steps:
  1. Optional: To display a report of floating license usage comparison, license usage, and license denials, complete the following steps:
    1. In the Admin Web UI, click the Server tab.
    2. In the Floating License Usage pane, click Reports .
    3. On the Floating License Usage page, from the drop-down menu, select a license type.
    4. Optional: In the Start date and End date fields, enter values.
    5. Click Show Reports .
  2. Optional: To display a list of issued Client Access Licenses on all servers that point to the floating license server, complete the following steps:
    1. In the Admin Web UI, click the Server tab.
    2. In the Floating License Usage pane, click Issued Leases .
    3. On the Issued Floating License Leases page, from the drop-down menu, select a license type.
  3. Optional: To display a list of issued Client Access Licenses on the current license server, complete the following steps:
    1. In the Admin Web UI, click the User Management tab.
    2. In the User Management pane, click Acquired Floating Licenses .
