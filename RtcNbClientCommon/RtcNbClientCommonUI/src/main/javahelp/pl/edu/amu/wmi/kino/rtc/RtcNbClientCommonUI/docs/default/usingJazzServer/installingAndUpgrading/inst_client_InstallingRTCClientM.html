Installing Rational Team Concert Client

  • Client installation overview
    There are two different ways to install the Rational Team Concert™ client.

    The client is distributed as a downloadable file via the IBM® Installation Manager.
  • Installing the client using a .zip file
    The client is distributed as a downloadable .zip file.
