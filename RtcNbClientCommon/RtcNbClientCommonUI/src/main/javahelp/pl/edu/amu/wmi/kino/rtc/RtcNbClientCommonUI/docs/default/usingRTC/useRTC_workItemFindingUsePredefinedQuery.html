Using a predefined query

Jazzâ„¢ provides a set of queries that you can use to retrieve work items that meet a common set of conditions.

You can use predefined quieries in

  • Netbeans
  • the Web interface

Using a predefined query in Netbeans

  1. On the RTC tab double click on Project's area name
  2. Navigate to Work Items > Shared Queries > Predefined
  3. Double-click a query to run it. The results appear in the Work Items view .
  4. To see the details of a work item, double-click it in the Work Items view . The work item opens in the Work Items editor .

Using a predefined query in the Web interface

  1. Click the Work Items link.
  2. In the navigation pane on the left, navigate to Shared Queries > Predefined.
  3. Click a query to run it. The results appear in the results pane on the right.
  4. To see the details of a work item, click it in the results pane. The work item opens in the work items editor.
