Modifying work items in a plan

In addition to modifying work items in the work item editor, you can modify work items in the Planned Items page of a plan. Modifying work items in a plan is more convenient because you do not have to open each work item in the work item editor to modify and save its details.
Before you begin
To view the work item in a plan, you must assign a milestone to the work item.
About this task
The table describes the tasks you can do with the work items in a plan:
Table 1. Modifying work items in a plan
To Do
Modify a work item in the work item editor Click the work item.
Modify a work item in the mini editor Hover the mouse pointer over the work item and click that is aligned in the same row as the work item. Click Save and Close after modifying the work item.
Demote or promote a work item Click to convert the plan to edit mode. Click to demote a work item or click the dropdown of the work item and select Demote or Promote. You can promote only demoted work items.
Move work items in a plan Click to convert the plan to edit mode. Select a work item that you want to move, click the drop-down list of a work item where you want to move work item to, and select Move Selected Element Above, Move Selected Element Below, or Move Selected Element Into
Add new work item Click and enter the work item details; or click to convert the plan to edit mode. Click the drop-down list beside the work item, and click New Work Item; and then select the work item type. Enter the work item details and click Save or Save and close.
