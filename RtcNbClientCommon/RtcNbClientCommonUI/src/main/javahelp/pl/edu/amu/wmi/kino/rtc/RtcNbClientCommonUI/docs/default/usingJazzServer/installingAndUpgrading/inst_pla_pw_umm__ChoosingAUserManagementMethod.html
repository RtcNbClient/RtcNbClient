Choosing a user management method

Choose one of the following user management options.
  • Managing users with Apache Tomcat user database
    The default user management enables you to manage users without any external user management system. Managing users with the Apache Tomcat user database is for Apache Tomcat installations only.
  • Managing users with Lightweight Directory Access Protocol (LDAP)
    This information helps you configure your LDAP registry to manage users.
  • Managing users with a custom realm
    Configure authentication and group membership in IBM® WebSphere® Application Server or Apache Tomcat.
