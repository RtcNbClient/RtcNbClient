Archiving a project

Important: Modifying project areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
You can archive a project from the administrative Web interface. Project archiving is permanent. You cannot unarchive a project.
Before you begin
You must be logged in to the Admin Web UI and be a member of the JazzAdmins group.
About this task
To archive a project:
  1. In the Admin Web UI, click the Project Area Management tab.
  2. On the Active Project Areas page, from the list of active projects, hover over a row and click the Archive icon.
  3. Click OK.
  4. Optional: To manage archived projects, in the left pane, click Archived Project Areas. On the Archived Project Areas page, select an archived project from the list.
