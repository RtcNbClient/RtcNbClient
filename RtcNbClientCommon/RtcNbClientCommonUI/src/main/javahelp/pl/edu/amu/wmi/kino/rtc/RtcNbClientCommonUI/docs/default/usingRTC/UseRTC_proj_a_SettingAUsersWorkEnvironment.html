Setting a user's work environment

Important: Modifying project and teams areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
Use the Work Environment page in the User editor to define a user's work location, work days and hours, and team assignments (by percentage). Set the time zone and regional settings for the user's work location to match the user's settings for Regional and Language Options, which are available in the Windows® Control Panel.
About this task

All users can modify their own Work Environment information.

To set the work environment for a user:

  1. In the Admin Web UI, click User Management.
  2. Click the user for which you want to set the work environment. To set the work environment for yourself, click your user name on the top right corner of the page.
  3. Click the Work Environment tab.
  4. Set the time zone and regional settings (language).
  5. In the Work Days list, hover the mouse pointer over a day and click
to modify the work day status and times.
  • In the Work Assignments list, click
to modify the percentage of the user's time assigned to the displayed teams. To add or remove teams for the user, open the team area and add or remove the user from the Members list.
  • Click Save.
