Managing license keys

You can use the administrative Web interface to manage server activation keys and client access licenses.
Before you begin
You must be logged in to the Admin Web UI and be a member of the JazzAdmins group.
About this task
To manage license keys through the administrative Web interface:
  1. In the Admin Web UI, click the Server tab.
  2. In the Configuration pane, click License Key Management .
  3. Optional: To upload a server activation key, complete the following steps:
    1. On the License Key Management page, in the Server Activation Key area, click Edit .
    2. In the Upload License Files wizard, in the Upload License Activation key field, click Browse and navigate to the server activation key location.
    3. Select the activation key.
    4. Click Finish .
      Note: The server activation key and corresponding client access license keys must be compatible. If you upload an incompatible server activation key, some or all existing client access licenses can become temporarily deactivated. After you upload the appropriate client access license(s), they will reactivate.
  4. Optional: To upload a client access license activation key, complete the following steps:
    1. On the License Key Management page, in the Client Access License Types area, click Add .
    2. In the Upload License Files wizard, in the Upload License Activation key field, click Browse and navigate to the client access license activation key location.
    3. Select the activation key.
    4. Optional: If the activation key uploaded requires that you specify the purchase count, in the Purchased field, enter the number of purchased licenses.
    5. Click Finish .
