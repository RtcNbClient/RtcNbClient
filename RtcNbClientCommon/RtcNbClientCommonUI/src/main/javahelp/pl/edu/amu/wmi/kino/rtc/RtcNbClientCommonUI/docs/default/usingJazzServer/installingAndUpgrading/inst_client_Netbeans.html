Installing Netbeans plugin for Rational Team Concert

To install:

  1. the latest version of RtcNbClient module. A .zip file will be downloaded and the .nbm files would need to be extracted.
  2. Open the Netbeans IDE.
  3. On the Menu Bar choose Tools and further choose the Plugins menu option.
  4. The Plugins dialog box opens up the Available Plugins tab by default; change the tab to Downloaded.
  5. Click the "Add Plugins..." button; A window pops up; navigate to the directory where you extracted .nbm files and add them. It shows up as visible in the figure.
  6. Click the install button on the Bottom-left corner of the dialog box.
  7. A confirmation window pops up; click Next to continue the installation.
  8. Next you need to read the license agreement and accept it. The RtcNbClient plugin is licensed under the GNU General Public License.
  9. Plugin is not digitally signed and you would need to ignore and continue inspite of the Validation Warning if you are confirmed about the source of the plugin.
  10. The plugin will install automatically.
  11. Once the installation is finished, you should restart the IDE.

The installation of the Plugin is complete and it can now be used with the IDE.
