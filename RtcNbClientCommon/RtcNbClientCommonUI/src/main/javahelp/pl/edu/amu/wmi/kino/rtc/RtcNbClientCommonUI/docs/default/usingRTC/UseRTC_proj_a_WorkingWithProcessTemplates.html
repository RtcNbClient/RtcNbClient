Working with process templates

Important: Modifying project and teams areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
Within the Web interface, you can edit process templates; import process templates from another Jazz™ Team Server; and export templates to another Jazz Team Server.
  • Exporting process templates
    In the Web interface, you can export and import process templates to and from the local file system. This enables teams to share their process templates with other Jazz Team Servers. Process templates define the process configuration, iteration structure, and all process attachments.
  • Importing process templates
    In the Web interface, you can export and import process templates to and from the local file system. This enables teams to share their process templates with other Jazz Team Servers. Process templates define the process configuration, iteration structure, and all process attachments.
  • Editing process templates
    You can modify process templates from within the Web interface.
