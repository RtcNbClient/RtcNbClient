Favorites folder

Here you can keep shortcuts to various resources to keep them at hand. In case you need quick access to work items use favorite folder.

To have quick access to work item:

  1. Add work item to favorites.
  2. Find work item easily by double-clicking on favorites folder.

To add work item to favorite folder:

  1. Double-click on Work Items folder.
  2. Navigate to choosen query.
  3. Double click on query.
  4. On work item view choose work item and right-click on it. Choose Add to favorites. Now work item should appear in favorites folder.
