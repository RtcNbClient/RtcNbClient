Work Item Folder

This folder includes a set of queries owing to you can easily retrieve proper work items.

This folder contains:

  • My queries folder : set of queries that are defined by user
  • Shared folder: set of queries that are shared between users
    • Predefined folder: Jazzâ„¢ provides a set of queries that you can use to retrieve work items that meet a common set of conditions.
    • Individually shared

You are able to

  • Find work items: Navigate to query. Double-click on it to open work items view. This view presents a list of work items that match query. For details see Work items view.
  • Add new work item : right click on Work Items and choose new work item.
  • Add new query: right click on Work Items and choose new query. It will be saved in My queries folder. For details see: Query editor
