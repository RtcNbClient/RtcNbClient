Adding and modifying users

Important: Modifying project and teams areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
You can add users to a project area or a team area. Project-level users can later be assigned to team areas after those teams are created. You can add users from other projects in the repository, import users from an external registry service such as an LDAP server, or create users. You can assign one or more roles to each user, based on roles that are defined in the process configuration. You can also designate the user as an administrator in a project area or a team area.
  • Importing users from an external registry service
    You can import users from an external registry service such as an LDAP server.
  • Assigning roles to a user
