Customizing the process in projects and teams

Important: Modifying project and team areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
You can set and modify project timelines; add and remove roles; and assign permissions for each role in a project area or team area.
  • Modifying permissions
    Permissions are assigned to individual roles at the project level or the team level in a project area, team area, or process template. You can modify the role's permissions for performing operations by using the Permissions tab of the project area or team area. You cannot modify permissions in the process template in the Admin Web UI.
  • Modifying the project timeline
    You can set or modify the start and end dates of the timeline for a project area.
  • Adding and removing roles
    You can add and remove individual roles for a project or team.
