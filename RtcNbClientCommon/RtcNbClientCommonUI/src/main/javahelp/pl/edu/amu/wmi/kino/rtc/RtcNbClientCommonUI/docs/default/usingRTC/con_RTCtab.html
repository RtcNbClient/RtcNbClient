RTC tab

Use this tab to work with yours Projects Areas. On this tab you see all Projects Areas that you are connected to.

Below you have description of its main funcionalities.

  • To Disconnect from Project Area : right-click on Project's Area name and choose Disconnect. After that projecy area will be removed from RTC tab. To re-connect use Services tab. Click Windows>Services. For details see : RTC Repositories
  • To find Work Items: double-click on Project's Area Name > Work Items. Next you are able to:
    • Use existing query : double-click on My queries or Shared Queries > Predefined folder and double click on query. Work items view with list of work-items will be opened. For details see: Work items view.
    • Create your own query : right-click on My queries folder and choose Add query. Now you will see a view where you can define yours query. For details see : Query editor.
    • Edit query : double-click on My queries or Shared Queries > Predefined folder and righ-click on query. Choose edit. Now you will see a view where you can edit query.For details see : Query editor.
  • Favorites folder : allows you quick acces to choosen work items. Double click on folder to see work items that you added to favorites. For details see: Favorite folder.
