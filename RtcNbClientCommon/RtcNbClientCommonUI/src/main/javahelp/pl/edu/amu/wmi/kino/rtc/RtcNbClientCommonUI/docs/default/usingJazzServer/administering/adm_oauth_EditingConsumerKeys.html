Editing consumer keys

You can edit an existing consumer key.
Before you begin
You must be a member of the JazzAdmins group.
About this task
To edit a consumer key:
  1. Click the Server tab.
  2. In the Configuration pane, click OAuth Consumer Management .
  3. On the OAuth Consumer Management page, in the Authorized Keys area, click on the Consumer Name of the key that you want to edit.
  4. In the Edit Consumer Key Properties dialog box, in the Consumer Name field, edit the values for the name, secret, or trusted values of the consumer key.
  5. Optional: To associate a pre-authorized user with the consumer key, which allows requests to be made by the user without additional authorization, click Select User . In the Select User dialog box, add the appropriate user.
    Note: Applications use this capability to run background administrative tasks, such as sending e-mail notifications, without user intervention.
  6. Click Finish .
