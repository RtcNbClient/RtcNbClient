Jazz Overview

Jazz is a scalable, extensible team collaboration platform for seamlessly integrating tasks across the software lifecycle. IBM® Rational Team Concert™ is built on the Jazz™ technology platform .
Tip: The Jazz technology platform is openly developed on , where you can register to participate in discussion forums, newsgroups, and the actual development process.
  • IBM Rational Team Concert quick reference
    If you want to dive right in and start exploring, this quick reference covers the key concepts, terms, and visual elements that you'll come across.
  • Overview and design goals of the Jazz technology platform
    Jazz is a scalable, extensible team collaboration platform for seamlessly integrating tasks across the software lifecycle.
  • Jazz technology platform architecture
    The Jazz technology platform is based on a client-server architecture. The Jazz Team Server , normally running on a secured server-class machine, hosts a set of services and a repository. Remote clients communicate with the Jazz Team Server over the network using HTTP.
  • Jazz kernel components
    The Jazz technology platform kernel includes the repository and team process components.
