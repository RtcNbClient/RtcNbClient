-dropTables

Repotools command used to drop the database tables.

Purpose

The dropTables command is used to drop the database tables.

Parameters

Attribute Description Required Default
teamserver.properties The file system path to the teamserver.properties file. No teamserver.properties
logFile Path to the log file. No repotools_dropTables.log
noPrompt Causes the database tables to be dropped without confirmation from the user. This option is useful when writing automated scripts. No N/A

Example

repotools -dropTables
