Modifying the project timeline

Important: Modifying project and teams areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
You can set or modify the start and end dates of the timeline for a project area.
Before you begin
You must be logged in to the Admin Web UI and be a member of the JazzAdmins group.
About this task
The project timeline defines a project area start and end date along with the iteration breakdown of those dates. To set or modify a project timeline:
  1. In the Admin Web UI, click the Project Management tab.
  2. Select a project area in the list of Active Project Areas.
  3. In the Details section, click Edit.
  4. In the Timeline field, click Edit. Enter or modify dates in the Start Date and End Date fields. Click Finish. If no timeline is set for the project, you see Create instead of Edit. Click Create and enter values in the Identifier, Display name, Start Date, and End Date fields. Click Finish.
  5. Click Save to save your changes to the project area.
