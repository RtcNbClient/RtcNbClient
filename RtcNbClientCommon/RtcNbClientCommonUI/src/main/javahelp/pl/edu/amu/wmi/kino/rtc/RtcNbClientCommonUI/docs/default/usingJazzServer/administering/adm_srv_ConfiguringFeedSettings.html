Configuring feed settings

You can configure the server database feed settings.
Before you begin
You must be logged in to the Admin Web UI and be a member of the JazzAdmins group.
About this task
To configure the server database feed settings:
  1. In Admin Web UI, click the Server tab
  2. In the Configuration pane, click Feed Settings .
  3. In the Feed Rendering Scheme field, type the rendering scheme to use for the feeds. Supported values are atom1.0 and rss2.0.
  4. In the Feed Query Maximum Entries field, type the maximum number of entries to include in any feed.
  5. In the Feed Cache Document Lifetime field, type the maximum length of time, in seconds, that a feed item can stay in the cache.
  6. In the Feed Entry Cache Size field, type the maximum number of feed items that the feeds can reuse from the cache.
  7. Click Save .
