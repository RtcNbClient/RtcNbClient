Grouping work items in a plan

Grouping work items is an easy way to view work items after combining the elements of Content, View Layout, and Color tabs of a plan mode. A plan mode is a single entity that represents the combination of these elements.
Before you begin
To view the work items of a plan, click Plans, and click a plan; and then click Planned Items.
About this task
Each plan type has a different set of plan modes. Select a plan mode from the View As drop-down list. You can also create new plan modes.

For more information about plan modes, see creating plan modes topic.

For more information about plan modes associated with a plan type, see the different plan types.

In addition to plan modes to group work items, you can also drag and drop work items in a plan. When you drag and drop work items the required attributes are simultaneously modified.
