Migrating the Jazz™ Team Server

This section of topics helps you change the application server and database configuration

The pre-configured Apache Tomcat server and Derby database enable you to get up and running quickly so you can explore the server and client features. Changing the application server or database can provide better performance and reliability in a production environment.

Migration consists of one or more of the following:
  • Changing from one application server to another. For example, migrating from the pre-configured Apache Tomcat server to a WebSphere® Application Server.
  • Changing from one database to another. For example, migrating from the embedded Derby database to a DB2® database.
