Getting connected

This lesson describes how to connect to a project area and a repository in Rational Team Concertâ„¢

The remainder of this tutorial is aimed at an individual developer.

  1. From the menu bar click Window > Services.
  2. On the Services tab right-click on RTC Repositories and choose Add new repository
  3. In the Create a Jazz Repository Connection dialog box:
    • Type or paste a server Location URI :
      Note: The Uniform Resource Identifier (URI) for an existing repository. Typically, this is in the form of http:/<server>:<port>/jazz Secure servers require the https:// protocol.
      Note: If the server is running on your computer, the server host name is localhost , and you can select the localhost URI from the dropdown list; for example, https://localhost:9443/jazz. The Location Name defaults to the server host name. You can modify it to simplify or clarify the name.
    • Type the user name and password that your Jazz administrator has provided for you.
    • Click Finish
  4. Under created repository connection you will see available project areas. Double click on selected project area to connect to it. RTC tab will be opened. Here you can work with project area that you just connected to.
