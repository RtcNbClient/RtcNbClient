Work items view

When you run a query, the Work Items view displays the results in a table where each row is a single work item

Administrators can customize the layout of work items view. To see details about work items attributes go to: Work items

Using work items view:

  • Sort the results according to column heading by clicking a column heading. Click the column heading again to change the sort order (ascending or descending).
  • Filter the query results. Right-click on one on work item attribute and choose Shows only rows where.
  • See details and edit work item. Righ-click on specified work item and choose Edit to open work item editor.

Toolbar icons let you perform the following actions:

  • Add New Query: to open query editor click
  • Edit query: to edit query click
  • Re-run the query you last run to get most current results.
