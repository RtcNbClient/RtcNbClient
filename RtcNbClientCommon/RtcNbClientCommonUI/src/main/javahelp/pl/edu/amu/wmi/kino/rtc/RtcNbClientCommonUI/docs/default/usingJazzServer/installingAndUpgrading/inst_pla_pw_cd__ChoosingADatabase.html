Choosing a database

This information addresses some considerations for choosing a database.
The Jazz™ Team Server includes Derby as the default database.
Note: Although a Derby database is included in the default configuration and requires no installation, this database cannot support more than 10 users. If you want to support more than 10 users and do not have access to an existing DB2 or Oracle database, install the bundled DB2 Workgroup Server edition.

Rational Team Concert™ Express, Standard, and Enterprise editions include a version of DB2 Workgroup Server Edition.

If you are already using DB2® or Oracle in your environment, the Jazz Team Server can be configured to run with your existing database software.
