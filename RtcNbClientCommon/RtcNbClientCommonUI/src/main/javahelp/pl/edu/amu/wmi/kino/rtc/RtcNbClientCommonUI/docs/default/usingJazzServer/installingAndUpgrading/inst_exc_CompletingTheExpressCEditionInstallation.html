Completing the Express-C edition installation

After the server is installed, you must consider some configuration options before continuing.
About this task
After the installation, there are a few additional actions to take.
  • Some files contain passwords. Those files should be protected so that they are readable only by users who are authorized to know the password for the accounts.
    • teamserver.properties - The Jazz™ Team Server requires that the database password is stored in JazzInstallDir/server/conf/jazz/teamserver.properties .

      When properties files are saved, the application always makes a backup copy of the previous version in the same directory. If you want to remove all files that contain the clear-text password, remove the backup properties files after configuring the server for the first time.

  • When connecting to the server with the Rational Team Concert™ Web browser, you might see security certificate warnings. To disable the warning, see Security certificates.
  • You are now ready to connect to the server with the Rational Team Concert Web browser.
  • Optional: Install the Build System Toolkit .
  • Look through the Getting Started topics in the online help. They contain helpful tasks for server administrators, team members, team leads and project administrators.
