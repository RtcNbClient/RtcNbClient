Archiving a user

You can archive or unarchive users from the administrative Web interface. When you archive a user, it releases the user's Client Access License, which allows you to reassign it to a different user.
Before you begin
You must be logged in to the Admin Web UI and be a member of the JazzAdmins group.
About this task
To archive a user:
  1. In the Admin Web UI , click the User Management tab.
  2. On the Active Users page, from the list of active users, hover over a row and click the Archive icon.
  3. Optional: To unarchive a user, complete the following steps:
    1. In the left pane, click Archived Users .
    2. On the Archived Users page, from the list of archived users, hover over a row and click the Restore icon.
