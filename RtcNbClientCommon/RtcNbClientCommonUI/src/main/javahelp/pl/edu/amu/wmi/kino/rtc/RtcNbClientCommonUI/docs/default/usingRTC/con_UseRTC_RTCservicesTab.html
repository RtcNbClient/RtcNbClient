RTC services

Here you start your work with Rational Team Concert in Netbeans:

Using RTC services Tab you are able to:

  • Add new repository connection.
  • Work with existing repository connections.
  • Connect/Disconnect from Project Areas.

New repository connection

To create new repository connection.

  1. Right click on RTC Repositories and choose Add new repository:
    • To create new repository connection you need:
      • Location URI of server
      • User name and password that your Jazz administrator has provided for you.
  2. Jazz Repository Connection window will be opened. For details see Add new repository wizard.

Working with existing repository connections

You are able to:

  • connect to repository: right-click on repository connection and choose Connect. Notice that when you are connected, repository icon changed from grey to coloured.
    If you want to connect to the repository automatically on start: right click on repository, choose edit and check Connect on start
  • disconnect from repository: :right-click on repository connection and choose Disconnect.
  • Delete repository connection: : right-click on repository connection and choose Delete.
    Warning: deleting repository connection does not require any confirmation.
  • Edit repository connection : right-click on repository connection and choose Edit. Window with connection details will be opened. For details see: Edit repository connection
  • Refresh repository connection: : to see the current connection status of repository. Right-click on repository connection and choose Refresh.

Connect/Disconnect from Project Areas.

Expand repository connection to see available Project Areas. Projects area grey icon means that you are not connected to Project Area.

  • Connect to Project Area: double click on selected Project Area. Project Area will open in RTC tab. For details see : Rational Team Concert Tab
  • Disconnect from Project Area: right-click on Project Area and choose Delete.
    Warning: disconnecting from project area does not require any confirmation.
