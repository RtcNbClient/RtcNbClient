Assigning default licenses to new users

You can assign default licenses to new users. The licenses are assigned to users that are imported from any external user directory, and to users that are imported or created through the Eclipse client, Web client, or LDAP nightly sync task.
Before you begin
You must be a member of the JazzAdmins group.
About this task
To assign a default license to a new user:
  1. Click the Server tab.
  2. In the Configuration pane, click License Key Management .
  3. On the License Key Management page, click Set Default .
  4. In the Set Client Access license key dialog box, select a default license.
  5. Click Finish .
