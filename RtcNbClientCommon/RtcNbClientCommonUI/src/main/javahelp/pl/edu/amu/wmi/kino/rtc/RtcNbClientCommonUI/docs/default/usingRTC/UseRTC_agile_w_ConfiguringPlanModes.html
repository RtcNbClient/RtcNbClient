Configuring plan modes

Important: Plans modifying plans is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
A plan mode consists of various elements that are on three tabs: Content, View Layout, and Colorize. You can change the elements of a plan mode to group work items according to your need.
Before you begin
You must have a plan and a plan mode.
About this task
To configure a plan mode:
  1. Open a plan in the Web UI.
  2. Click
.
  • You can do the following:
    Tabs Description
    Content Select the elements of Style, Grouping, Filtering, Sorting, and Bars in the way you want to group work items.
    View Layout Displays different attributes of a work item as columns in the Planned Items page. Add the columns that you want to view in the Planned Items page. To view a user-defined column in this tab, you must first define the work item attribute; and then in the Plan Attributes editor, click Add.
    Colorize Colors the work items based on expressions. Click Add to add a color with label.
    • To change color, click on the color cell.
    • To change the label, click on the label cell.
    • To add an expression, click on the expression cell.
    Type an expression to color the work items. For example, you can type priority:high to color work items whose priority is set to high. If you use multiple conditions for different attributes, the work items that match the all the conditions are colorize. For example, for priority:medium severity:major , the work items that has both priority set to medium and severity set to major are colorized.

    If you use multiple conditions for the same attribute and if both the conditions are matched, the work items that match the first condition are colorized. For example, for priority:high priority:low , the work items with priority set to high are colorized.
  • Click Save.
