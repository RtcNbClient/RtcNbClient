Logging in to the Admin Web UI

You can administer the Jazz™ Team Server through the Web interface.
Before you begin
The team server must be started.
About this task
To log in to the Admin Web UI:
  1. In a supported Web browser, on the Jazz Team Server workstation, navigate to the administrative page .
    Note: To remotely access the Jazz Team Server administrative page, replace localhost with the name or IP address of the Jazz Team Server workstation.
  2. When prompted, type a user name and password. For Tomcat installations, you can use the following default values:
    1. Default user name: ADMIN .
    2. Default password: ADMIN .
  3. To check the server status, on Status Summary page, complete the following steps:
    • Under Server Status , verify that the database status is connected.
    • Under Service Error Summary , verify that there are no service errors.
