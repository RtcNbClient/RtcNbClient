My queries and shared folder.

Thanks to My queries folder and Shared folder you can easily retrieve proper work items.

To work with work items and queries you can choose between:

  • My queries folder - this folder contains a set of queries that are defined by user.
  • Shared queries folder - this folder contains a set of queries that are shared between users. In this folder queries are grouped in Predefined queries folder and Individually shared folder.
    Jazzâ„¢ provides a set of predefined queries that you can use to retrieve work items that meet a set of conditions.

To find work items:

  1. Double click on folder.
  2. Double click on query.
  3. Work items view will be opened. This view presents a list of work items that match query. For details see: Work Items View.

Using My queries and shared folder you are able to:

  • Edit existing query : double-click click on folder, right click on query and choose Edit. Query editor will be opened. For details see : Query Editor

Using My queries folder you also able to :

  • Add new query: right-click on My queries folder and choose Add query. Query editor will be opened. For details see : Query Editor
  • Rename existing query : double-click on My queries folder then select query and right click on query and choose Rename.Enter new name in the pop-up window.
  • Delete existing query: double-click on My queries folder then select query and right click on query and choose Delete.
    Warning: deleting queries does not require confirmation.
