Create plan

Important: Creating plans is not available in Netbeans plugin for Rational Team Concert.

Before creating a plan, you must create a project area with at least one timeline, iteration, and a team area associated with a work item category. Also, you must have permission to save a plan. For more information about permission, see Modifying permissions topic.

To create a plan:

  • In the Web UI, click Plans .
  • Click Create Plan .
  • Type a plan name and select Team Area, Iteration, and Plan Type.
    The Plan Type can be a release plan or an iteration plan.
  • Click Save .
