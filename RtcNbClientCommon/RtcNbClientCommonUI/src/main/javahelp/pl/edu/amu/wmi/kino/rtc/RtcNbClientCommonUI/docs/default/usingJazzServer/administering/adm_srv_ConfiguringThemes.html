Configuring themes

You can change the look and feel of the Web UI by using themes with custom CSS, HTML, and JavaScript code. You can upload, preview, and activate themes for all users on the server.
Before you begin
You must be a member of the JazzAdmins group.
About this task
To configure theming:
  1. Click the Server tab.
  2. In the Configuration pane, click Themes .
  3. On the Themes page, complete the following fields:
    • In the Label field, type a theme label.
    • Beside the Zip field, click Browse and navigate to a theming zip file.
    • Click Upload .
  4. Recommended: To preview the theme, beside the theme name, click Preview .
  5. Click Activate .
