Connecting to the Repository

A repository is a central location to store data. Users must create a connection from the Rational Team Concert™ client to a repository for access to project areas and work with the project artifacts. Your Jazz™ administrator must supply information to enable your access to the repository.

You can connect to the repository using:

  • Netbeans
  • the Web interface

Connecting to the Repository from Netbeans

Important: This topic assumes that your Jazz administrator has created a repository and a project area.
  1. Create Repository Connection
    • Click: Window > Services
    • Right click on RTC Repositories and choose Add new repository
  2. Jazz Repository Connection window
    • Type or paste a Location URI :
      Note: The Uniform Resource Identifier (URI) for an existing repository. Typically, this is in the form of http:/<server>:<port>/jazz Secure servers require the https:// protocol.
      Note: If the server is running on your computer, the server host name is localhost , and you can select the localhost URI from the dropdown list; for example, https://localhost:9443/jazz. The Location Name defaults to the server host name. You can modify it to simplify or clarify the name.
    • Type the user name and password that your Jazz administrator has provided for you.
    • Click Finish
  3. Under created repository connection you will see available project areas. Double click on selected project area to open RTC tab and work work with it.

Connecting to the repository from a Web browser

You can use a web browser to access a Rational Team Concert repository.

To access a Rational Team Concert repository through a Web browser:

  1. Open your Web browser and go to the URL for your Jazz Team Server Web interface . The URL is similar to the following:https://localhost:9443/jazz/web
  2. Log in: type your's user ID and password and click Prelude from the list of projects.

You can now browse the Prelude project area using the various pages of the Web UI.
