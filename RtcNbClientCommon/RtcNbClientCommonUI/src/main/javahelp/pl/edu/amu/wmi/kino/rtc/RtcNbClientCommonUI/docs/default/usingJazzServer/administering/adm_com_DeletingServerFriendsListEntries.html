Deleting server friends list entries

You can delete entries from the server friends list.
Before you begin
You must be a member of the JazzAdmins group.
About this task
To delete an entry from the server friends list:
  1. Click the Server tab.
  2. In the Configuration pane, click Cross-Server Communication .
  3. On the Cross-Server Communication page, in the Server Friends List area, view the list of servers established for cross-server communication.
  4. To the right of the entry to be deleted, click Delete Friend .
  5. Confirm that you want to delete the friend.
