Shared Queries

This folder contains a set of queries that are shared between users. Owing to those queries you can easily retrieve proper work items. They are grouped to Predefined queries folder and Individually shared folder.

To find work items:

  1. Double click Shared Queries folder
  2. Double click Predefined or Individually shared folder.

  3. Double click on query.
  4. Work items view will be opened. This view presents a list of work items that match query.

  • Edit existing query: Navigate to Predefined folder and double-click on it.Select query, right-click on query and choose Edit.
