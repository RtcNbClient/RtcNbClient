Importing users from an external registry service

Important: Modifying project and teams areas is not available in Netbeans plugin for Rational Team Concert. Following information relates to web user interface.
You can import users from an external registry service such as an LDAP server.
Before you begin
You must belong to the JazzAdmins repository group to perform this task.
About this task
To create a user in a project area or a team area:
  1. In the Admin Web UI, click the User Management tab.
  2. On the Active Users page, click Import Users.
  3. Type a partial match for a name in the external registry. Begin with a space to search last names.
  4. Select a name from the results and click Select. Optionally enter a search string for another user to import.
  5. When you finish selecting users, click OK.
What to do next
After you finish importing users, be sure to assign client access licenses to them.
