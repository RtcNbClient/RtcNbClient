Overview and design goals of the Jazz technology platform

Jazz is a scalable, extensible team collaboration platform for seamlessly integrating tasks across the software lifecycle.
Tip: The Jazz™ technology platform is openly developed on , where you can register to participate in discussion forums, newsgroups, and the actual development process.
The Jazz technology platform is designed and built to:
  • Support seamless integration of tasks across the software lifecycle.
  • Facilitate team collaboration and coordination throughout the software lifecycle.
  • Help teams build software more effectively.
  • Support globally distributed development teams.
  • Provide solutions scalable from small teams up through large enterprises.
  • Maintain audit trails and automate bookkeeping.
  • Support UI integrations (IDE, web browser, etc.) that fit the needs of customers.
  • Foster a broad ecosystem of tool providers, including independent software vendors (ISVs).
  • Make software development more enjoyable.

The Jazz technology platform 's principal role is to provide tool writers with mechanisms to use, and rules to follow, that lead to seamlessly-integrated lifecycle tools. These mechanisms are exposed via well-defined APIs. The Jazz technology platform also provides useful building blocks and frameworks that facilitate developing new tools.
